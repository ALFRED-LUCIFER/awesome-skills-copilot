## AI Agent System

This organization uses a shared GitHub Copilot agent system managed in the `awesome-skills-copilot` repository, packaged as the **`awesome-skills-copilot` VS Code Copilot plugin** (v2.0.0).

* **15 Avenger-themed agents** — Jarvis (orchestrator), Friday (backend), Karen (frontend), Vision (reviewer), E.D.I.T.H. (Playwright E2E), Mainframe, Hope, Veronica, H.O.M.E.R., Dum-E, and the hidden workers (U, Butterfingers, Droney, C.L.U.E.)
* **17 slash commands** — quick actions for developers, POs, and scrum masters
* **13 portable skills** — reusable knowledge packages for code generation
* **3-tier guardrails** — safety, quality, and compliance enforcement
* **Plugin install**: `Chat: Install Plugin From Source` → paste `awesome-skills-copilot` Git URL

See the [Developer Manual](COPILOT-DEVELOPER-MANUAL.html) in `awesome-skills-copilot` for full setup and usage instructions.
