# confluence-content-guide

> Reference guide for structuring AI-consumable Confluence documentation with page templates, labeling conventions, and NG Space taxonomy.

## Purpose

Defines the standard structure for all Confluence pages in the NG Space — ensuring consistency for both human readers and AI agents. Provides page templates, labeling conventions, and content organization rules.

## When to Use

- Creating or validating Confluence content
- Establishing NG Space taxonomy
- `@docs-writer` / `@docs-planner` agent workflows

## Key Features

- Page structure templates (Engineering Standards, ADRs, System Design, Testing Strategy, DevOps, Runbooks)
- Labeling convention with `ai-consumable` tag for AI-readable pages
- Content organization taxonomy for the NG Confluence Space

## Used By

- `@docs-writer` agent
- `@docs-planner` agent
