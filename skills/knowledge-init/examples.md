# Knowledge Init — Templates & Examples

> Loaded on-demand by `token-budget-loader` at full tier only.

## Invocation Modes

| Command | Behaviour |
|---------|-----------|
| `/knowledge-init` | Full scan — create all missing AGENTS.md and docs/ files |
| `/knowledge-init [domain]` | Scope to one domain folder only |
| `/knowledge-init --update` | Refresh only stale domains (from drift log) |
| `/knowledge-init --meta-only` | Update project metadata in INDEX.md only |

## Discovery Commands

### .NET domains
```bash
find {domain}/ -name '*.cs' | grep -E 'Controller|Service|Repository|Entity|Dto|Event|Command|Query|Constants' | head -30
```

### React / TypeScript domains
```bash
find {domain}/ -name '*.ts' -o -name '*.tsx' | grep -E 'Controller|Service|hook|api|query|mutation|type|interface|constant|slice|store' | head -30
```

### General folder patterns
```bash
# Feature-based
ls -d src/features/*/ src/modules/*/ src/pages/*/ 2>/dev/null | sort
# DDD
ls -d src/Domain/*/ src/*/Domain/ 2>/dev/null | sort
# Monorepo
ls -d apps/*/ libs/*/ packages/*/ 2>/dev/null | sort
# Exclude noise
EXCLUDE="node_modules|dist|build|obj|bin|.git|Migrations|wwwroot|coverage|test|spec"
```
