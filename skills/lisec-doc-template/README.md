# myorg-doc-template

> Complete HTML template, compliance rules, and per-section content guide for Org product documentation. GPSR/PLD/CRA compliant.

## Purpose

Provides the canonical HTML template, CSS corporate design variables, and section-by-section content guide for generating Org product documentation. Ensures compliance with GPSR, PLD, and CRA regulations.

## When to Use

- `@product-manual` agent generating documentation
- Creating new product manuals
- Updating existing product documentation

## Template Sections (A–I)

| Section | Content |
|---------|---------|
| A | Product Overview |
| B | Getting Started / Installation |
| C | Features & Functionality |
| D | Configuration |
| E | User Interface Guide |
| F | API Reference |
| G | Troubleshooting |
| H | Compliance (GPSR/PLD/CRA) |
| I | Version History |

## Pre-generation Checklist

- Product name, version, type
- i18n key locations
- Route definitions
- Feature list from Jira/Confluence

## Used By

- `@product-manual` agent
